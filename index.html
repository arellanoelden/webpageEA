Projects:

  • Planit Triton

    • Planit Triton is a site I helped create last year for a class competition. What this site does is create a hub of information for students for their classes. We bring together the schedule of classes and course catalog along with which professors are teaching said course for that quarter. When a class is selected the student can view class info such as location and description, grade distributions for that course,ratemyprofessor ratings, caperatings and links to comments for that professor. This way, students can compare classes and professors in order to best suite their needs. In order to achieve this my group had to scrape a large amount of data from several sites. I helped mainly on the scraping but also aided in building the database and UI.

    This project created a want in me to learn more about databases and I did so a few quarters later when I took an offical databases course. This project was also one of my first introductions into Python as I had to learn webscraping using BeautifulSoup in Python in order to be able to grab the information we needed.

  • Triton Calendar

    • While not as large of a project I still have rather enjoyed this project as it has taught me valuable skills. This site takes in a student's username and password for their student account, asks for permission from their google calender,and automatically imports their class schedule into their calenders. This is useful because every quarter I know several students who spend tedious time inputing each course with the name of the course,time,location etc. This webapp does everything in mere seconds. I vastly enjoyed this project because I gained more experience in web design along with learning how to gain permissions from other accounts such as Google and Tritonlink accounts.
