Front end experience

Here is my relatively simple webpage but I have used some good techniques to ensure that this site is a good example of how I develop. First off is speed, a site needs to be fast. This site is also a progressive web app(PWA) where files get cached and on mobile devices they can be installed directly to the device. In many of my other projects seen on github I also add progressive web apps and also have experience using React, Angular, and creating accelerated mobile pages(AMP). The client experience is crucial and often times developer make moves that help them on the developer experience such as using a js library but that can cause a much larger download to the user. Many forget to optimize images,change resolutions, lazy load images and so much more to increase the user experience. Look and feel are also important, things like material UI libraries can still be good especially if served locally, minimized or done through a cdn but in the end one should ensure the user experience is as high as can be.
